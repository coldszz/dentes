# DobryZub как использовать
шаг 1. запустите сборку вручную
перейдите на вкладку Actions репозитория

слева выберите "Build Laravel for Beget / build"

нажмите кнопку "Run workflow" → "Run workflow"

дождитесь окончания сборки (зелёная галочка)

шаг 2. скачайте готовый архив
На вкладке Actions нажмите на последний запущенный workflow

внизу найдите раздел Artifacts

скачайте файл dentes-beget.zip

внутри этого архива — полностью готовый проект с папкой vendor и всеми зависимостями. 

шаг 3. залить на хостинг бегет

далее все распаковываем в бегет и настраиваем .env под свой сайт и генерируем ключ (key.php) чтобы вставить в APP_KEY=UR_KEY. 

устанавливаем права bootstrap/cache и stogare 755 

и создаем или скачиваем бд, чтобы все работало

сайт должен заработать
